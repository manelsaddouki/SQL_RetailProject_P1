# SQL_RetailProject_P1

## Retail Sales Data Analysis

This is a small project/exercise designed to practice and implement data analytics skills. The project focuses on cleaning, exploring, and analyzing a dataset related to retail sales to extract meaningful insights and patterns.

## Project Overview

The aim of this project is to apply data analytics techniques to a retail sales dataset. The analysis covers data cleaning, exploratory data analysis (EDA), and generating insights that could support business decisions.

  ### Dataset
  
  The dataset contains information on retail sales transactions, including:
  
    * Sale date and time
    * Product category
    * Quantity sold
    * Total sale amount
    * Customer information 
    Note: All data used is synthetic or anonymized for educational purposes.


Conclusion

This project is a practical exercise to enhance skills in data cleaning, exploration, and analysis, while demonstrating the ability to draw meaningful insights from retail data.
